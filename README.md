# Minimal Text Editor

A simple text editor built with React and Vite.

## Quick Start

```bash
# Install
npm install

# Run
npm run dev
```

## Available Commands

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run linter

## Tech Stack

- React 18
- Vite
- ESLint
